# Release Notes

## v1.0.0-beta.1 - 2026-04-22

这是 AIVibe OS 的第一版公开 Beta。

### Included

- 跨工具 root rules：`AGENTS.md`
- Claude 入口：`CLAUDE.md`
- Cursor 入口：`.cursor/rules/` 和 `.cursor/commands/`
- AIVibe commands：`/spec /plan /build /test /review /code-simplify /ship`
- 25 个 root skills
- Claude 项目级 skills 和 subagents
- Figma 到代码 Top能力
- 治理脚本与固定 closeout 输出

### Positioning

- 这是一个可分发、可复用、可落地的 AI 开发操作系统 Beta
- 它不是零配置的全自动出货平台
- 它更适合做团队开发底座和项目试点产线

### Known Limits

- 仍然需要在真实项目里继续验证高强度闭环
- 发布、部署、观测、真实业务回放还需要继续接入更多项目证据
- 目前适配层以 Claude Code 和 Cursor 为主

### Recommended Next Step

- 选一个真实项目，用 AIVibe OS 跑第一次完整交付闭环
