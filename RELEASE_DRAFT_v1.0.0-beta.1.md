# AIVibe OS v1.0.0-beta.1

这是 AIVibe OS 的第一版公开 Beta。

## What It Is

AIVibe OS 是一套面向真实产品交付的人机协同开发操作系统。

它不是一个单独的万能 Agent，也不是一堆散的提示词，而是一套把 AI 开发从“更快写 Demo”推进到“可验证、可回滚、可复用、可出货”的系统。

## Included In This Release

- cross-tool root rules
- 7 个标准 commands
- 25 个 root skills
- Claude / Cursor 适配层
- Claude 项目级 skills
- Claude 专家会诊 subagents
- Figma 高保真还原能力包
- QA / 发布 / 回滚模板
- 治理脚本与固定 closeout 输出

## Why This Exists

大部分 AI coding 体验，停在：

- 能写一点代码
- 能做一点页面
- 能跑出一个 demo

但真正困难的是：

- 需求怎么定
- 任务怎么拆
- 质量怎么守
- 发布怎么做
- 回滚怎么留

AIVibe OS 解决的是这条完整链路。

## Current Status

这是 `v1.0.0-beta.1`。

意味着：

- 已经可以正式试用
- 已经可以对外分发
- 已经具备一套完整骨架

但它还不是：

- 零配置全自动出货平台
- 不需要真实项目回灌的最终形态

## Best Fit

AIVibe OS 最适合：

- 小团队 / 创业团队
- 想把 AI 开发方法系统化的人
- 想统一 Claude / Cursor / Codex 工作流的人
- 想建立 commands / skills / adapters / governance 的团队
