# Changelog

## 1.0.0

首次发布正式版 `Figmatocode1.0`。

这版不是简单搬运，而是一次重构式整理。

### 这版做了什么

- 建立新的正式目录 `Figmatocode1.0/`
- 重新定义单一权威入口 `README.md`
- 明确核心工作流、复杂页闭环和关键层参数工作流
- 保留真正有执行价值的模板
- 保留复杂页案例和轻量页案例
- 把打包分享说明收进 1.0，保证目录本身可独立发出使用
- 把组件映射层降到扩展区，避免干扰首次使用
- 让目录本身自洽可用

### 这版的主线

`Figma 真值 -> 页面分型 -> 结构拆层 -> 原素材接入 -> 参数收敛 -> 浏览器截图对稿 -> 工程验证`

复杂页增加：

`关键层参数表 -> 分层实现`

### 这版不是做什么

- 不是全自动产品
- 不是一键生成页面工具
- 不是自由设计包

### 后续可能的 1.1 方向

- 增加第二个非 Splash 案例
- 给浏览器验收补更稳定的自动化脚本链路
- 补多端适配和状态矩阵模板
- 真正把组件映射层接进页面落地流程
