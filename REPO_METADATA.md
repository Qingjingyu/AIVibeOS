## Public Positioning

AIVibe OS 不是一个单独的 AI Agent。

它是一套可复用的开发操作系统，负责把：

- 需求定义
- 任务拆解
- 增量实现
- 测试验证
- 代码评审
- 发布与回滚

固定成一条稳定的 AI 协作产线。

## Audience

适合：

- 小团队或创业团队
- 想把 AI 开发从 Demo 推进到交付的人
- 想统一 Claude / Cursor / Codex 工作方式的团队
- 想建立自己的 commands / skills / agent workflow 的团队

不适合：

- 期待零配置全自动出货的人
- 不愿意维护规范、验证和回滚的人

## Key Message

AI 开发真正缺的不是“更强的模型”，而是“更稳定的出货系统”。

AIVibe OS 解决的是后者。
